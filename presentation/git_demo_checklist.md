# Git Demo Checklist

- Show clean repository title and README overview.
- Show folder structure: `src`, `sql`, `presentation`, `data`, and `outputs`.
- Open `src/run_pipeline.py` to show the full pipeline order.
- Open `src/preprocess.py` to show feature engineering.
- Open `sql/sample_queries.sql` to show required SQL analysis queries.
- Open `outputs/figures` locally to show generated chart files.
- Open `outputs/reports/daily_insights.md` locally to show GenAI-style report.
- Open `presentation/chart_usage.md` to show why each figure belongs in the slide deck.
- Run `python -m src.validate_outputs` before presenting.
- Mention that raw data, processed data, database, charts, and reports are generated artifacts.
- Keep the live dashboard portion to one ticker unless an evaluator asks for comparison.
- If pushing to GitHub, commit source files and docs, then optionally add selected screenshots to a `screenshots/` folder.
